Gira

    Gira is a fullstack website created for a college club that exists in several universities in Mexico. In this website, users can publish articles with images, create events with images and videos, send massive emails to mailing lists, accept donations through PayPal's API, and register for events that this club holds. In addition, users can view posted classes and courses made available by Gira's administrative team. Gira's User Interface was designed by Daniela Martinez, a graphical designer, following the website requirements set by the team and myself.
